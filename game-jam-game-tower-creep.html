GMTK Game Jam (Tower Creep)

Game Jam made using Unity and in C# - 16/08/2024

The game was created for the GMTK game jam and it is a 2D semi-tower defence game where you
run around and try to survive as you place towers and upgrade them and explode the enemy towers and stop the enemies from spawning.
The game has a story line too and you can play the original game uploaded for the game jam here.











I updated this game after the game jam to fix some of the main issues and reuploaded it here.
this has a few changes like the explosives explode the enemy towers and there are now enemy towers!
There is also a lot of particle effects and camera
shakes for things. So the game is a lot harder.
